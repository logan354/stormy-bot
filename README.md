# Stormy
The best weather bot for New Zealand on Discord. Using the MetService API.

## Commands
- **Weather**: `forecast`, `observation`, `severe-weather-outlook`, `thunderstorm-outlook`, `warning`
- **Utility**: `help`, `invite`, `ping`, `statistics`
